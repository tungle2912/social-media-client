## Check node version

```bash
node -v
```

## Get updated Node version

Install [nvm](https://github.com/nvm-sh/nvm) for nodejs version management

Use latest stable version

```bash
nvm install latest
```

```bash
nvm use latest
```

Use specific version

```bash
nvm install 20.13.1
```

```bash
nvm use 20.13.1
```
