## Run local invironment
- Run localhost
```bash
npm install
npm run dev
```

- Fix prettier
```bash
npm run prettier:fix
```

- Run lint check and fix
```bash
npm run lint
npm run lint:fix
```

- Run coverage
```bash
npm run test
npm run test:watch
```
